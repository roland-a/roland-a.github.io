Personal Projects

Infinibiome

Page

This is the first large project I have created. It is a mod for the game Minecraft that adds infinitely varied landscapes. Most other terrain mods I tried only add in multiple predefined biomes. These predefined biomes tend to have little variations within and do not seamlessly transition between biomes. I wanted to make a terrain generator that makes every area feel unique and seamless. This is why I created Infinibiome. Instead of predefined biomes, every world feature such as amplitude, slope, temperature, humidity, etc., are varied and independent.

ML-Craft

Page

This is another Minecraft mod that is in development. After being frustrated with procedure terrain generation with Infinibiome, and after seeing the recent bloom of AI technology, I came up with a new idea. What if I were to use generative AI to generate good-looking terrain? I decided to have this idea be my senior design project, leading a team of two others. After weeks of hard work, we came up with a working proof concept with these requirements:

  • The generated world size can be any arbitrary size
  • The model must generate elevation data that are very similar to the real-life elevation it's trained on
  • Multiple models can work seamlessly together to create multiple biomes within a generated world.

For our proof of concept, we decided to use two models: one based on a real mountain and another based on a real desert. First, we have an arbitrary biome map. Yellow represents deserts, and gray represents mountains:

Then we use our desert model to generate deserts where it's yellow on the map, and our mountain model to generate mountains where it's gray:

MLCraft is still in its infancy. There are still a lot of new features that need to be added before it can be officially released.

Here is a roadmap of the features that will be added next:

  • Create a working proof of concept.
  • Have the generated elevation data be playable in Minecraft.
  • Add oceans and beaches to the biome-map generator.
  • Use better-trained models.
  • Add more models.
  • Move the models from Python to Java.
  • Have the elevation data be generated on the Create World screen.
